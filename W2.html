ImageJ Macro Recap

What does this code do?

for(i = 0; i < nSlices; i++){
    print(i);
}
nSlices and print are built in functions

Improved solution


// Open input window to enter input file
#@ File (label = "Input directory", style = "file") input
open(input);
// Or: open('/Downloads/test/example1.tif');

// Count number of slices in image
for(i = 1; i <= nSlices; i++){
    print("Slice: " + i);
}
print("Image contains: " + nSlices + " slices");
#@ indicates a script paramter

Macro Generalisation

You might need to generalise your macro. For example, if you merge channels, you might have a command like 

run("Merge Channels...",
    "red=[Edges of boats.gif] green=boats.gif blue=boats.gif gray=*None*");

How can you make this generic?

title = getTitle();
run("Merge Channels...",
     "red=[Edges of " + title + "] green=" + title + " blue=" + title + " gray=*None*");

Measurements

Making measurements, what to measure, line vs area

  • Open any image. You can download the example shown in the slides: 02-Biological Image.tif
  • Select Line Selection tool (fifth tool from left)
  • Draw a line (shift sets 45° increments)
  • Run [Analyze > Plot Profile]
  • Select rectangle Selection tool (first tool from left)
  • Draw a rectangle (shift / ctrl modify)
  • Hit 'm' or run [Analyze > Measure]

The measurements provided are set via [Analyze > Set Measurements...] except for selection-specific measurements (length, angle, coords)

The ROI manager is used to store and measure selections

  • Make a selection (any type)
  • Hit 't' or run [Edit > Selection > Add to Manager]
  • Repeat several times
  • Run [More > Multi-measure]
  • Use [More > Save] for better data provenance!

Applications: Batch Processing

Why batch process? File conversion, batch processing, scripting

Manual analysis can be laborious, error prone and not provide the provenance required. Batch processing allows the same processing to be run on multiple images. The built-in [Process > Batch] menu has lots of useful functions.

  • Download this zip file to the desktop and unzip it. It contains a subset of dataset BBBC008 from the Broad Bioimage Benchmark Collection to try it out.
  • Create an empty folder on your desktop called Output
  • Try [Process > Batch > Convert...]
  • Try [Process > Batch > Macro...]

Use the macro recorder [Plugins > Macros > Record] and The Batch Macro function to:

  • Set the scale of the image to 0.1 microns per pixel
  • Draw a 10μm green scale bar in the bottom left corner
  • Write the filename onto each image (in red) ()
  • Print out the filename and spot count to the log ()
